Before running you need to:
	1}Install Kinect SDK 2.0 (KinectSDK-v2.0_1409-Setup)
	2)Install Matlab Compiler Runtime 2015a or compatible (MCR_R2015a_win64_installer)
	3)Set up a database on MySQL server (v57) 



To use emulator copy directory Emulator to directory where application expects records.
